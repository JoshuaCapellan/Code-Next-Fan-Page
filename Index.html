What is Code Next?

Code Next is a program at Google where kids are taught about coding and computer science in order to prepare them for a life in the coding business itself. By teaching kids about coding this early in life, they are given the necessary tools to succeed in the computer science business.

Why is Code Next so benefical?

Provides children with a basic knowledge of coding, so that they are better prepared for a job in the field of computer science.

By accepting only students of color, it takes steps towards diversifying the computer science community.

It allows for students to immersify themselves into the world of computers, which also allows for them to develop a genuine interest in computers, the internet, and how they work.
